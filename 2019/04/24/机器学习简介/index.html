感觉当时上机器学习课的时候学得似懂非懂，现在补一补课吧。

分类

  • 机器学习有很多分类，主要可以分为监督学习（supervised learning）和无监督学习（unsupervised learning）。
  • 监督学习是指教计算机怎么学习，无监督学习是指让计算机自己学习。
  • 监督学习可以分为分类问题(classification problems)和回归问题(regression problems)，分类问题对应于离散的情况，而回归问题对应于连续的情况。对于只有少量离散值的问题，应该将它作为分类问题。
  • 在无监督学习中，所给的数据没有标签或者标签都相同，常使用聚类算法(clustering algorithm)，将数据集划分为不同的簇；另一个使用无监督学习方法的问题时鸡尾酒会的音频分离问题。

    线性回归(linear regression)

  • 线性回归属于监督学习方法中的回归问题，它是用用假设函数(hypothesis function)来预测y是x的线性函数。

    特征缩放(Feature Scaling)

  • 经过特征缩放后梯度下降法可以尽快迭代至全局最小点。
  • Get every feature into approximately a range。实际可取的范围可以大于+1，或者小于+1，但也别太大，比如+100，但也不能太小，比如+0.001。

    matlab使用

  • length()命令返回矩阵的行和列最高维度，不过一般常用于向量。
  • load()命令用来加载数据。
  • who命令用来显示当前工作区有哪些变量。
  • whos命令用来显示工作区变量的具体信息。
  • save+文件名+要保存的变量用来将变量保存到文件中。
  • 想要将变量保存为人可见的形式，可以使用命令：save 文件名.txt 变量 -ascii。
  • size(A) = [3,2]，给A矩阵添加一列，例： A = [A, [100; 101; 102]]。
  • size(A) = [3,2]，size(B) = [3,2], C = [A, B]， 则size(C) = [3, 4]，相当于把B加在A的右边。
  • size(A) = [3,2]，size(B) = [3,2], C = [A; B]， 则size(C) = [6, 2]，相当于把B加在A的下边。
  • max(a), a是一个行向量，则可以得到该行向量中的最大值，可以使用命令[val, ind] = max(a)，其中val用来返回最大值，ind用来返回最大值在行向量中的下标；也可以直接使用val = max(a)来返回a向量中的最大值。
  • max(A)，A是一个矩阵，则max(A)会返回A中每一列的最大值。
  • a = [1, 15, 2, 0.5]，find(a < 3) = [1 3 4]，即返回a中所有小于3的元素的索引。
  • max(A, [], 1)可以得到A矩阵的每一列的最大值。max(A, [], 2)可以得到A矩阵每一行的最大值，max默认求的是每一列的最大值。
  • 如果要求矩阵A中所有元素的最大值，则可以使用max(max(A))，或者将矩阵A变为向量再求最大值，即max(A(:))。
  • flipud(A)表示将A矩阵垂直旋转。
  • 可以使用逗号连续调用函数。
  • 可以使用addpath()命令来添加搜索路径。
